### Key Decisions:

1. **Only ONE repo matters**: `newlens-uk/newlens-v2-preview` (main branch)
2. **Only ONE workspace**: `~/.openclaw/workspace/newlens-v2-preview/`
3. **Never clone again** - always work in existing directory

5. **Always pull before push** to catch conflicts early

```bash

git pull origin main
# ... make changes ...

```
