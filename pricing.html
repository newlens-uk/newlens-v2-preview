What Do These Terms Mean?

Workflow

A single automation that connects your tools and performs a task. Example: "When tenant enquiry arrives, create folder in Google Drive, send welcome email, add to CRM."

Execution

Each time a workflow runs counts as one execution. If your "tenant enquiry" workflow runs 50 times this month, that's 50 executions.

Node

Each step in a workflow (e.g., "check Gmail", "create Drive folder", "send Slack message"). More nodes = more complex workflow. We count nodes to keep workflows maintainable.

Change Time

Included minutes per month for us to modify your workflows (adjust a filter, change an email template, add a condition). Major new workflows or logic rebuilds are additional.

VPS (Virtual Private Server)

A cloud server you rent from providers like Hetzner or DigitalOcean (¬£10-20/month). We guide you through setup (5 minutes), then we install and manage n8n on it. You own it forever.

Setup Fee

One-time cost to discover your processes, design workflows, build + test automations, train your team, and go live. Waived for alpha partners.
